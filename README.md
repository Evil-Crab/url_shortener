# Url Shortener
##### Available at: [http://188.226.141.44](http://188.226.141.44)
### Requirements
  - Python >= 3.6
  - tornado
  - redis
  - validators


### Installation
```sh
 pip3 install https://github.com/Evil-Crab/url_shortener/zipball/master
 ```
